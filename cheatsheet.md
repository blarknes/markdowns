# markdown tags cheat sheet:

````txt
Title (from h1 to h6):
# -> ######

Hyperlink:
[hypertext](https://url/)

Image:
![fallback name](./image.png)

Block:
```type of text (e.g. py; js; sh)
$ cd folder/subfolder
```

Italic:
_This text is italic_ or *This text is italic*

Bold:
**This text is bold**

Unordered list:
- item#1
- item#2

Ordered list:
1. item#1
2. item#2

Note:
> This is a note

Table:
| table | header |
| --- | --- |
| table | data |
| table | data |

Horizontal line:
---
````
